## HABITS TRACKER APP

# Project description
- This is a Single Page Application which allows users to add new habits, view all their habits, track whether each habit is active or inactive, and view detailed information about each habit.

# Features
- View a list of all habits on the home page.
- Add a new habit using a form with controlled inputs.
- Click on a habit to view its full details.
- Track whether a habit is active or inactive.
- Navigate between pages without refreshing the browser.
- Responsive layout that works on desktop, tablet, and mobile screens.
- Conditional rendering for loading, error, and empty states.

# Technologies used
- React (v18)
- React Router DOM 
- JavaScript 
- CSS 
- JSON Server 
- Vite 

# Setup Instructions
1. Clone the repository to your local machine.
2. Install dependencies using : npm install.
3. Start the JSON Server using npm run server in the terminal.
4. Start the development server using npm run dev.

# Routes

# Known Challenges
1. Setting up React Router and making sure the correct route matched the correct page took some debugging.
2. Managing shared state across components required careful planning of where to place useState and which props to pass down.
3. Connecting the form submission to JSON Server and reflecting the new habit in the list without a page refresh needed useEffect to re-fetch after a POST request.
4. CSS layout on smaller screen sizes required extra media queries to keep the navbar and cards readable.

# Future Improvements
- Add a habit completion tracker where users can mark a habit as done for the day.
- Add a streak counter that tracks how many consecutive days a habit has been completed.
- Add the ability to edit or delete an existing habit.
- Add filtering so users can view only active or only inactive habits.
- Add user authentication so each person can have their own private habit list.
- Improve the UI with progress indicators or charts to visualize habit consistency.
